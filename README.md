## Objective
This project demonstrates the use of setState() in Flutter to dynamically update the UI.

## Features
- Counter increments on button press
- Background color changes randomly
- Emoji mood changes based on counter value
- Reset button to restart the app

## How setState() is Used
setState() is used to:
- Update the counter value
- Change background color
- Change emoji
- Rebuild the UI dynamically
